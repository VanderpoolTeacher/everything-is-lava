Angled Ground System

Terrain Cross-Section (side view)

Implementation: Replace flat GROUND_Y with getGroundY(worldX) function:

  • Base: GROUND_Y - (worldX * tan(15°)) ≈ GROUND_Y - (worldX * 0.267)
  • Clamped so ground never goes above ~40% of screen height
  • All collision checks use getGroundY() instead of GROUND_Y constant
  • Scroll offset still increments by RUN_SPEED horizontally — the angled ground creates the climb feel
