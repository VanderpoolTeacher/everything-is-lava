Section 4 of 4: Background, Level Flow & Integration

Destroyed City Background

Parallax Layers (back to front)
Far: ruined skyline (0.1x)
Mid: crumbled buildings (0.35x)
Near: rubble & embers (0.8x)
Ground: volcanic rock (1.0x)
  • Sky: Dark red/black gradient with ash particles drifting down (purely cosmetic)
  • Far layer (0.1x): Jagged ruined skyline silhouettes — broken rooftops, collapsed towers
  • Mid layer (0.35x): Crumbled buildings with faint lava glow from cracks. Occasional flickering orange windows (fires still burning)
  • Near layer (0.8x): Rubble, broken walls, ember particles rising. Replaces the neon signs from 1-x
  • Ground (1.0x): Dark volcanic rock with lava vein cracks glowing orange. Follows the 15→25° incline
  • Smoke wisps: Semi-transparent gray particles rising from random ground points

Level Flow & Difficulty Curve

First Third (0-1000f)
  • Incline: 15°
  • Lava rain: moderate (every ~40f)
  • Rocks: very rare (every ~150f)
  • Trees: mostly medium, well-spaced
  • Player learns the slope + trees
Second Third (1000-2000f)
  • Incline: ~20°
  • Lava rain: increasing (every ~30f)
  • Rocks: occasional (every ~100f)
  • Trees: mixed sizes, some burning
  • Pressure builds
Final Third (2000-3000f)
  • Incline: ~25°
  • Lava rain: heavy (every ~20f)
  • Rocks: frequent (every ~70f)
  • Trees: more tall ones, many burning
  • Peak difficulty before level end

Integration with Existing Systems

  • Level numbering: Internally level === 4, displayed as "Level 2-1"
  • Duration: 3000 frames (50 seconds)
  • Transition in: Standard level transition screen showing "Level 2-1"
  • Transition out: Standard levelComplete state → ready for 2-2
  • Vehicles: None — rocks replace them as the horizontal threat
  • Streetlamps: None — trees replace them as platforms
  • Konami code: Press 4 after code to jump to 2-1
  • Score: Continues formula: levelTimer/3 + 3000 base (level 4 bonus)
  • Music: Reuse existing track for now — placeholder until new tracks added
  • Lives: Carry over from 1-3 (or start fresh if entering via Konami)
