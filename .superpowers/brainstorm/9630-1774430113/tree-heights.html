Tree Heights & Platforming

Trees replace streetlamps as the main platform. How should heights vary?

A

Uniform Height (like lamps)

All trees are the same height (~135px, matching lamp posts). Consistent platforming — players know exactly how high to jump. Simple and predictable.

B

3 Fixed Sizes (Small / Medium / Tall)

Small: ~80px (single jump). Medium: ~135px (double jump, same as lamps). Tall: ~180px (double jump from medium tree or from higher ground). Creates varied vertical gameplay.

C

Random Range

Heights vary randomly between 80-180px. Every tree is a different challenge. More chaotic and unpredictable — keeps players on their toes but harder to read at a glance.

Note: The issue says "trees of varying heights" — so some variation is expected. Option B gives variety while keeping it readable. Option A would deviate from the spec.
