Burning Tree Timer

When lava hits a tree, how long before it collapses?

A

Quick Burn (~2 seconds / 120 frames)

Trees burn fast — barely enough time to land and jump off. High tension. Forces quick reactions and discourages lingering on any platform. Very punishing if you're mid-jump when it ignites.

B

Medium Burn (~3.5 seconds / 210 frames)

Enough time to land, assess, and jump to the next platform. The fire is a warning — you have a window but can't idle. Feels fair while still creating urgency. Visual flames escalate as the timer counts down.

C

Slow Burn (~5 seconds / 300 frames)

Generous window. The tree burns visually but stays solid for a while. More forgiving — burning trees are an environmental detail that eventually removes platforms rather than an immediate threat.

Context: At RUN_SPEED 2.5px/frame, a tree stays on screen for roughly 6-7 seconds. A 3.5s burn means the tree could collapse while still visible — adding visual drama. A 2s burn means some trees vanish almost immediately after being hit.
